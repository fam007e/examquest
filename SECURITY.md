# Security Policy

## Supported Versions

Currently, the following versions are supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 2.0.x   | :white_check_mark: |
| 1.x     | :x:                |

## Reporting a Vulnerability

If you find a security vulnerability, please do NOT open an issue. Instead, please email the project maintainer or use the contact information provided in the profile.

We will acknowledge your report within 48 hours and provide a timeline for a fix if applicable.
