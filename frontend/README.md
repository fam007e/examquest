# ExamQuest Frontend

This directory contains the user interface of ExamQuest. It is built using React 19, styled with Tailwind CSS v4, and bundled with Vite 7 using the Rolldown compiler.

## Dev Stack
- **Framework**: React 19
- **Bundler**: Vite 7
- **Styling**: Tailwind CSS v4
- **State & Transition**: Framer Motion
- **Icons**: Lucide React

## Getting Started
To run the development server independently:
```bash
npm install
npm run dev
```

The app will start at `http://localhost:5173`.
